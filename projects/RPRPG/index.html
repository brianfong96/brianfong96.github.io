R.P.R.P.G.

An offline password generator that creates reproducible passwords from memorable inputs, without storing secrets or sending data anywhere.

Back to Projects
Description

R.P.R.P.G. combines personal inputs with a deterministic algorithm so you can regenerate a service-specific password whenever you need it.

  • No passwords are stored or transmitted.
  • Works fully offline in a modern browser.
  • Changing the account field gives you a different password for each service.
  • You can tune the character set, length, and extra passphrase for stronger outputs.
