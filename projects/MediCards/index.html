How to add a new deck

  1. Create a folder for your specialization under projects/MediCards/.
  2. Add an index.html that loads a specialization-specific data script and the shared deck logic.
  3. Populate a <Specialization>.js file that sets window.mediCardsDeck with cards, explanations, and sources.
